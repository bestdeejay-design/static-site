## 1. ВВЕДЕНИЕ

### 1.1 Зачем нужен раздел выпускников

Приложение UniverID ориентировано не только на действующих студентов, но и на выпускников института. Раздел «Выпускник» позволит:

- Поддерживать связь с институтом после окончания обучения
- Участвовать в мероприятиях для выпускников
- Следить за новостями и достижениями института
- Вносить вклад в развитие института (эндаумент-фонд)
- Находить выпускников для профессиональных контактов
- Быть в курсе встреч и AEHO-мероприятий

### 1.2 Целевая аудитория

- Выпускники СПбГТИ всех лет (бакалавры, специалисты, магистры, аспиранты)
- Действующие студенты старших курсов (для предварительного ознакомления)

### 2.2 Механизм переключения

**Вариант А: Выбор при входе**
- На экране «Открыть демо» добавить переключатель: «Я выпускник»
- При выборе — показывается интерфейс выпускника

**Вариант Б: Смена статуса в профиле**
- В профиле студента добавить опцию «Я выпускник»
- При активации — переход в режим выпускника

**Вариант В: Отдельный вход**
- Создать отдельную страницу входа для выпускников
- URL: `?mode=alumni` или отдельная кнопка

**Рекомендуемый вариант:** Вариант А (выбор при входе) — проще в реализации и понятнее для пользователя

## 3. СТРУКТУРА РАЗДЕЛА «ВЫПУСКНИК»

### 3.1 Главная страница выпускника

**Компоненты:**
1. **Приветствие** — «Добро пожаловать, выпускник [Имя]!»
2. **Год выпуска** — указывается в профиле
3. **Быстрые действия**:
   - Мой профиль
   - События
   - Встречи выпускников
   - Эндаумент-фонд
   - Доска почёта
   - Профессиональная сеть
4. **Лента событий** — адаптированная (без студенческих событий)

### 3.2 Профиль выпускника

**Поля:**
- Фото (загружаемое)
- ФИО
- Год выпуска
- Специальность / направление
- Текущее место работы (optional)
- Должность (optional)
- Компания (optional)
- Email
- Телефон (optional)
- LinkedIn (optional)
- О себе (краткое описание)

**История карьеры (новое):**
- Место работы 1: Компания, должность, годы
- Место работы 2: Компания, должность, годы
- ...

**Настройки уведомлений:**
- Встречи выпускников: вкл/выкл
- Новости института: вкл/выкл
- Эндаумент-фонд: вкл/выкл
- AEHO-мероприятия: вкл/выкл

### 3.3 События для выпускников

**Типы событий:**

1. **Встречи выпускников**
   - Ежегодные встречи (юбилеи выпусков)
   - Тематические встречи (по специальностям)
   - Встречи с выдающимися выпускниками
   - Встречи с руководством института (ректор, проректоры)

2. **AEHO-мероприятия** (Ассоциация выпускников)
   - Конференции
   - Мастер-классы
   - Нетворкинг-мероприятия
   - Экскурсии по институту

3. **Мероприятия института**
   - День открытых дверей (помощь в организации)
   - Олимпиады (судейство)
   - Конференции (модераторство)

4. **Карьерные события**
   - Ярмарки вакансий (работадатель)
   - Менторство студентов

**Интерактивность событий (новое):**

Каждое событие имеет кнопку «Я иду»:
- ✅ Записаться (подтвердить участие)
- ❌ Отменить запись
- 👥 Показать количество записавшихся

**Для института:**
- Видеть количество записавшихся
- Планировать загрузку площадки
- Отправлять напоминания

### 3.4 Эндаумент-фонд СПбГТИ

**Что это:**

Эндаумент-фонд — это целевой капитал, который работает на постоянной основе для финансирования института. Деньги не расходуются, а инвестируются, и доход от инвестиций направляется на цели фонда.

**Примеры использования (по данным СПбГУ):**
- Стипендии для студентов
- Обновление оборудования
- Поддержка научных исследований
- Мероприятия и программы
- Обновление инфраструктуры

**Интерфейс эндаумент-фонда:**

1. **Информация о фонде**
   - Общее описание: что такое эндаумент
   - Цели фонда
   - Прозрачность (ссылки на отчётность)

2. **Текущие программы**
   - Название программы
   - Цель сбора
   - Собранная сумма / целевая сумма
   - Прогресс-бар
   - Описание программы

3. **История пожертвований**
   - Список крупных жертвователей (по желанию)
   - Общая сумма сбора

4. **Сделать пожертвование**
   - Кнопка перехода на страницу оплаты
   - Реквизиты / ссылка на платёжную систему

**Примеры программ:**
- «Стипендии для талантливых студентов» — 5 000 000 ₽ / 10 000 000 ₽
- «Обновление лаборатории химии» — 2 000 000 ₽ / 5 000 000 ₽
- «Фонд развития спорта» — 500 000 ₽ / 1 000 000 ₽

### 3.5 Доска почёта

**Разделы:**

1. **Выдающиеся выпускники**
   - Фото
   - ФИО
   - Год выпуска
   - Достижение
   - Краткая биография
   - Ссылка на полную статью

2. **Почётные доктора ТИ**
   - (уже есть в разделе «Об институте»)

3. **Лауреаты премий**
   - Государственные премии
   - Премии института
   - Международные награды

4. **Успешные компании выпускников**
   - Название компании
   - Отрасль
   - Год основания выпускником
   - Количество сотрудников

### 3.6 Профессиональная сеть

**Функционал:**

1. **Каталог выпускников**
   - Поиск по имени, году выпуска, специальности
   - Фильтры: отрасль, должность, компания

2. **Профиль выпускника (публичный)**
   - Фото
   - ФИО
   - Год выпуска, специальность
   - Текущая должность и компания
   - Опыт (кратко)
   - Контакт (по согласованию)

3. **Сообщества**
   - Выпускники года X
   - Выпускники факультета Y
   - Профессиональные группы (IT, химия, менеджмент...)

4. **Сообщения**
   - Личные сообщения между выпускниками
   - (упрощённая версия чата)

    event.registered = !event.registered;

  }
}
