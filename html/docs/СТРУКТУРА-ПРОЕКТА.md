# Структура проекта UniverID

> Мобильное PWA-приложение для студентов СПбГТИ (Технологический институт)
> Обновлено: май 2026

---

## 1. ГЛАВНАЯ СТРАНИЦА (home)

**Файл:** `js/app.js` → `renderHome()`

**Содержимое:**
- Dashboard с основными показателями (средний балл, часы, текущий семестр)
- Счётчик дней до сессии
- Модули быстрого доступа (6 иконок): Оценки, Расписание, Клубы, Заметки, Навигатор, О институте
- Карточки быстрого доступа: Оценки, Расписание, Сообщения
- Кнопка "Сессия" (расписание экзаменов/зачётов)
- Блок "Об институте" под каруселью

## 3. ОЦЕНКИ (grades)

**Файл:** `js/app.js` → `renderGrades()`

**Данные:** `appData.transcript`

**Содержимое:**
- Группировка по семестрам
- Для каждой дисциплины:
  - Название дисциплины
  - Часы
  - Тип контроля (экзамен/зачёт)
  - Оценка (5, 4, зачтено)
  - Дата

**Примеры данных:**
- Семестр 1: Математика, Физика, Информатика, История России
- Семестр 2: Математика, Программирование, Английский язык, Физкультура
- Семестр 3 (текущий): Алгоритмы и структуры данных, Базы данных, Теория вероятностей, Сети и телекоммуникации

## 4. РАСПИСАНИЕ (schedule)

  - Название дисциплины
  - Тип (лекция/практика/лабораторная)
  - Аудитория
  - Преподаватель

## 17. СИСТЕМНЫЕ ФУНКЦИИ

### Тема (темизация)
- `toggleTheme()` — переключение светлая/тёмная
- `loadTheme()` — загрузка из localStorage

### Навигация
- `showPage(name)` — переход на страницу
- `goBack()` — возврат назад
- `openApp()` / `closeApp()` — открыть/закрыть приложение

### Подписки (новое)
- `toggleCategory(id)` — включить/выключить категорию событий
- `toggleClub(id)` — вступить/выйти из клуба
- `appData.profile.subscriptions` — хранение подписок

### Сохранение состояния
- localStorage для:
  - `univerid_lastPage` — последняя страница
  - `univerid_history` — история навигации
  - `univerid_appOpen` — приложение открыто
  - `univerid-theme` — тема
