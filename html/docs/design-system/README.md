# Дизайн-система — Основы

Это стартовый набор токенов и компонентов для UniverID demo.

Структура
- css/main.css — токены и базовые компоненты
- pages/rewrite/_header.html — header
- pages/rewrite/_footer.html — footer

Рекомендуется при добавлении новых стилей использовать переменные из :root в css/main.css.
