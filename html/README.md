# UniverID — Цифровая экосистема Технологического института

## 📋 О проекте

UniverID — это комплексная демо-версия цифровой экосистемы университета, разработанная для Технологического института.

**Сайт:** https://bestdeejay-design.github.io/univerid/  
**Домен:** univerid.ru

## 🎯 Основные модули

### ✅ Реализовано в демо:

1. **Главная страница (лендинг)**
   - Презентация экосистемы
   - Модули и возможности
   - Статистика
   - Контакты

2. **Авторизация**
   - Демо-вход
   - Госуслуги (заглушка)
   - ЛК студента (заглушка)
   - Корпоративная почта (заглушка)

3. **Дашборд студента**
   - Профиль студента
   - Статистика
   - Уведомления
   - Навигация по модулям

4. **Расписание**
   - Расписание на неделю
   - Типы занятий (лекции, практики, лабораторные)
   - Преподаватели и аудитории

5. **Успеваемость**
   - Электронный журнал
   - Средний балл (GPA)
   - Оценки по предметам

6. **Карьера и стажировки**
   - Вакансии от IT-компаний
   - Фильтрация по типу
   - Отклик в 1 клик

7. **Цифровое резюме**
   - Профиль студента
   - Навыки
   - Проекты
   - Достижения

8. **Льготы и скидки**
   - Цифровая студенческая карта
   - Категории льгот (IT, транспорт, еда, развлечения)
   - Партнёрские скидки

9. **Мероприятия**
   - Афиша событий
   - Регистрация
   - Информация об организаторах

10. **AI-ассистент**
    - Чат-бот
    - Быстрые вопросы
    - Интеграция с модулями

## 🛠 Технологии

- **HTML5** — семантическая разметка
- **CSS3** — дизайн-система, CSS variables, grid, flexbox
- **JavaScript (ES6+)** — интерактивность
- **GSAP** — анимации
- **GitHub Pages** — хостинг

## 🚀 Запуск локально

1. Клонируйте репозиторий:
```bash
git clone https://github.com/bestdeejay-design/univerid.git
cd univerid
```

2. Откройте `index.html` в браузере или используйте Live Server

3. Для демо перейдите на `/pages/index.html`
